Diabetes Predictor

A Machine Learning Web App, Built with Flask, Deployed using Heroku.





enter '2' for 00 to 59 (Low)

enter '3' for 60 to 80 (Normal)

enter '0' for 81 to 90 (Elevated)

enter '1' for 91 or above (High)




enter '1' for values between (16-166) which is a Normal range

enter '2' for any other value.



enter '3' for values between under 18.5 (Underweight)

enter '0' for values between 18.5 to 24.9 (Healthy)

enter '2' for values between 25.9 to 29.9 (Overweight)

enter '1' for values between 30.0 and above (Obese)
